# Ubuntu Home Lab Foundation

A two-system Ubuntu home lab built to support hands-on learning in Linux, Docker, remote administration, cloud engineering, and security.

## Overview

This project documents the initial buildout of my home lab environment using two Ubuntu-based systems:

- **PC1:** Ubuntu Desktop with XRDP enabled for remote administration
- **PC2:** Ubuntu Server configured as a Docker host and managed through Portainer

The goal of this lab is to create a practical environment for building cloud, DevOps, and security projects.

## Current Lab Architecture

PC1 - Ubuntu Desktop
- XRDP remote access
- Management workstation

PC2 - Ubuntu Server
- SSH access enabled
- Docker installed
- Portainer deployed
